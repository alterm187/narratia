# Book Cover Images

Add your book cover images here with these exact names:

- mirrors-cover.jpg (600x900px recommended)
- stick-carrot-cover.jpg (600x900px recommended)
- minds-reflection-cover.jpg (600x900px recommended)

Until you add real covers, the site will show placeholder letters (L, ≈Å, O).

## How to Add Images

1. Save your book covers with the names above
2. Copy them to this directory
3. Commit and push:
   ```bash
   git add public/books/*.jpg
   git commit -m "Add book cover images"
   git push
   ```

The site will auto-update on Vercel!
